HTML 5 Features

Semantic Tags

HTML 5 introduced several new semantic tags that help to describe the type of content between the opening and closing tags. These tags include:

  • header
  • nav
  • article
  • section
  • aside
  • footer

Multimedia Elements

HTML 5 introduced several multimedia elements that make it easier to incorporate audio and video content into a web page. These elements include:

  • audio
  • video
  • source
  • track

APIs and Data Storage

HTML 5 introduced several new APIs and data storage options that make it easier for developers to create more interactive web pages.

  • canvas
  • webgl
  • indexeddb
  • websqldatabase
